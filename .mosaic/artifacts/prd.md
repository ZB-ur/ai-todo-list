## Goal

构建一个简洁美观的 Web 端计算器应用，支持基础四则运算，使用 React + Tailwind CSS 实现，作为纯前端组件无需后端依赖。

## Features

- **basic-arithmetic**: 支持加(+)、减(-)、乘(×)、除(÷) 四则运算，支持小数点输入，支持连续运算（如 1+2+3=）
- **display**: 单行显示屏，展示当前输入的数字或运算结果，超长数字自动缩小字号适配显示区域
- **clear-backspace**: C 按钮清除全部输入恢复初始状态；⌫ 按钮删除最后一个字符
- **keyboard-layout**: 标准计算器布局：0-9 数字键、四则运算符、等号(=)、小数点(.)、C、⌫，采用 4 列网格，0 键占两格
- **safe-expression-parser**: 实现安全的数学表达式解析器（禁止使用 eval()），正确处理运算符优先级（先乘除后加减）
- **error-handling**: 除以零时显示 "Error"，用户可按 C 清除错误状态继续使用
- **responsive-layout**: 响应式设计，在桌面端居中展示，移动端全宽适配，按钮尺寸适合触控操作（最小 48x48px）

## Constraints

- 纯前端实现，不依赖任何后端服务或数据库
- 技术栈限定为 React + Tailwind CSS
- 禁止使用 `eval()` 或 `new Function()` 解析数学表达式
- 单次运算数值范围：JavaScript Number 安全整数范围内（±9007199254740991）
- 浮点精度：显示结果最多保留 10 位有效数字，消除常见浮点误差（如 0.1+0.2 应显示 0.3）

## Out of Scope

- 科学计算功能（三角函数、对数、幂运算等）
- 计算历史记录
- 主题切换 / 暗色模式
- 括号支持
- 单位换算
- 键盘快捷键支持
- 数据持久化（localStorage 等）
- 多语言 / 国际化
