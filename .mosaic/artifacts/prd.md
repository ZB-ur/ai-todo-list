## Goal
提供一个开箱即用的 Web 端简单计算器，支持四则运算，界面简洁、响应快速、零学习成本。

## Features
- **basic-arithmetic**: 支持加（+）、减（−）、乘（×）、除（÷）四则运算，点击等号（=）显示计算结果
- **display**: 顶部单行显示区域，实时显示当前输入的数字或运算结果，超长数字自动缩小字号
- **number-pad**: 标准九宫格数字键盘（0-9），底部 0 键占两格宽，布局符合传统计算器习惯
- **decimal-point**: 支持小数点输入，每个数字最多一个小数点，重复点击无效
- **clear**: AC（All Clear）按钮清除所有输入并重置为初始状态 0
- **backspace**: 退格按钮删除最后一位输入字符，删空后显示 0
- **chained-operations**: 支持连续运算（如 1 + 2 × 3），每次输入新运算符时先计算前一步结果并显示
- **error-handling**: 除以零时显示 "Error"，点击 AC 或任意数字键可恢复正常状态
- **responsive-layout**: 响应式布局适配桌面与移动端，移动端按钮最小触控区域 44×44px

## Constraints
- 纯前端实现，无后端依赖
- 技术栈：React + Tailwind CSS
- 运算精度使用 JavaScript 原生 Number 类型（不引入高精度库）
- 显示区域最大支持 16 位数字，超出后显示科学计数法
- 不使用 `eval()` 或 `Function()` 进行表达式求值，需自行实现计算逻辑

## Out of Scope
- 科学计算功能（三角函数、对数、幂运算等）
- 计算历史记录
- 键盘快捷键输入
- 主题切换 / 暗黑模式
- 单位换算
- 括号与运算优先级（仅支持从左到右顺序计算）
- 本地存储 / 状态持久化
- 国际化 / 多语言
