## Goal
打造一款面向学生、工程师和专业人士的跨平台（iOS/Android）科学计算器 App，以拟物化 3D 质感浅色界面为核心差异化，支持竖屏基础运算和横屏完整科学计算两种布局模式。

## Features
- F-BASIC-CALC: 基础四则运算 — 支持加、减、乘、除、括号嵌套、百分比运算，遵循标准运算符优先级
- F-SCI-TRIG: 三角函数计算 — 支持 sin、cos、tan 及其反函数 asin、acos、atan，支持 DEG/RAD 模式切换
- F-SCI-LOG: 对数与幂运算 — 支持 ln、log（以10为底）、x²、xⁿ、√x、ⁿ√x、EXP
- F-SCI-CONST: 数学常量与阶乘 — 内置 π、e 常量一键输入，支持 n! 阶乘运算
- F-EXPR-DISPLAY: 实时表达式显示 — 显示区同时展示当前输入的完整表达式和实时计算结果，支持连续运算
- F-PORTRAIT-LAYOUT: 竖屏基础布局 — 竖屏模式下展示数字键盘、基础运算符和常用科学函数的精简键盘
- F-LANDSCAPE-LAYOUT: 横屏科学布局 — 横屏模式下展开完整科学键盘，按键分为数字区、运算符区、科学函数区三个清晰分区
- F-NEUMORPHIC-UI: 拟物化 3D 质感界面 — 浅色背景配合柔和阴影、3D 凸起按键效果、按压状态反馈动画，所有按键满足 WCAG AA 对比度标准（≥ 4.5:1 文字对比度、≥ 3:1 非文字元素对比度）
- F-DEG-RAD: 角度/弧度模式切换 — 提供 DEG/RAD 切换按钮，当前模式在显示区常驻指示

## Constraints
- 目标平台为 iOS 和 Android 移动端，设计稿需覆盖两种平台的屏幕尺寸规范
- 交付物仅为 UI 设计稿（竖屏+横屏两套布局）和计算引擎 API 规范，不包含代码实现
- UI 采用拟物化/Neumorphism 风格，浅色现代配色方案；所有按键必须满足 WCAG AA 无障碍对比度标准
- 表达式解析引擎 API 需支持标准运算符优先级和括号嵌套
- 横屏布局需明确按键分区：数字区、运算符区、科学函数区，确保高频操作效率

## Out of Scope
- 计算历史记录功能
- 单位换算功能
- 图形绘制 / 函数图像
- 公式编辑器 / Math Notes
- 云同步 / 跨设备数据同步
- 主题切换（深色模式等）
- RPN（逆波兰表达式）模式
- 矩阵运算、微积分、统计回归等高级数学功能
- Apple Watch / 平板专用布局
- 代码实现与可运行原型
