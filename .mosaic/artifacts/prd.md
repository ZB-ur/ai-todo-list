## Goal

提供一个纯前端的简单计算器 Web 应用，支持四则运算，交互清晰、即开即用。

## Features

- **F1 — 数字输入**：用户可通过点击按钮输入 0-9 数字和小数点，输入实时显示在屏幕上
- **F2 — 四则运算**：支持加（+）、减（−）、乘（×）、除（÷）四种运算，点击运算符后继续输入第二个操作数，点击等号（=）显示结果
- **F3 — 连续运算**：计算结果可作为下一次运算的第一个操作数，用户无需重新输入即可继续计算
- **F4 — 清除与退格**：提供 AC（全部清除，重置为初始状态）和 ⌫（退格，删除最后一位输入）两个按钮
- **F5 — 错误处理**：除以零时显示 "Error"，用户点击 AC 可恢复；数字溢出时以科学计数法显示
- **F6 — 显示屏**：顶部显示区域展示当前输入和计算结果，字体大小随位数增多自动缩小以避免溢出
- **F7 — 响应式布局**：在桌面端和移动端均可正常使用，移动端按钮尺寸适配触摸操作（最小 44×44px 触摸区域）

## Constraints

- 纯前端实现，不依赖后端 API
- 使用 React + Tailwind CSS 构建
- 禁止使用 JavaScript `eval()` 进行表达式求值，必须用状态机或逐步求值方式实现
- 每次只支持两个操作数的单步运算（非括号表达式），通过连续运算链式计算
- 数字精度：使用 JavaScript 原生 Number 类型，显示结果最多保留 10 位有效数字

## Out of Scope

- 括号与复合表达式（多运算符优先级）
- 科学计算功能（三角函数、对数、幂运算等）
- 键盘快捷键支持
- 计算历史记录
- 主题切换 / 暗黑模式
- 单位换算
- 离线 PWA 支持
