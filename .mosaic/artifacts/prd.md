## Goal
提供一个基于 Web 的简单计算器，支持基础四则运算，采用经典网格布局，用户零学习成本即可使用。

## Features
- **display**: 单行显示屏，展示当前输入数字或运算结果，超长数字自动截断/缩小字体
- **digit-input**: 0-9 数字键输入，支持多位数连续输入
- **decimal-point**: 小数点输入，同一数字中仅允许输入一次小数点
- **basic-operations**: 四则运算（加 `+`、减 `-`、乘 `×`、除 `÷`），点击运算符后输入下一个操作数
- **equals**: 等号键执行当前运算并显示结果，结果可作为下一次运算的第一个操作数
- **chained-operations**: 连续运算支持，输入 `3 + 5 × 2` 时按下 `×` 即先计算 `3 + 5 = 8`，再以 8 为基数继续
- **clear**: 清除键（AC）重置计算器到初始状态，显示归零
- **divide-by-zero**: 除以零时显示错误提示（如 "Error"），用户可按 AC 重置

## Constraints
- 纯前端实现，使用 React + Tailwind CSS，无后端依赖
- UI 采用标准计算器网格布局（4 列），包含 0-9、四个运算符、小数点、等号、AC
- 所有运算使用 JavaScript 浮点数，不引入任意精度库
- 显示区域最多展示 12 位数字，超出时用科学计数法或缩小字体
- 组件通过 Playwright 截图验证 UI 输出

## Out of Scope
- 科学计算功能（三角函数、对数、幂运算等）
- 历史记录功能
- 键盘快捷键支持
- 括号与表达式编辑
- 单位换算
- 主题切换/暗黑模式
- 后端 API 或数据持久化
