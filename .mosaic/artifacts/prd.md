## Goal

构建一个基于 Web 的简单计算器应用，提供即开即用的四则运算体验，支持键盘和鼠标双模式输入，具备计算历史记录功能。

## Features

- **basic-arithmetic**: 支持加（+）、减（-）、乘（×）、除（÷）四则运算，支持小数和负数输入
- **expression-display**: 顶部显示当前输入的完整表达式，底部显示实时计算结果预览
- **keyboard-input**: 支持键盘数字键（0-9）、运算符（+、-、*、/）、Enter（=）、Backspace（删除）、Escape（清空）快捷操作
- **button-input**: 提供标准计算器按钮布局（0-9、+、-、×、÷、=、C、⌫、.、±），支持鼠标/触摸点击
- **calculation-history**: 保留最近 20 条计算记录（表达式 + 结果），支持点击历史记录回填结果到当前输入，页面刷新后历史记录通过 localStorage 持久化
- **decimal-precision**: 使用精度处理方案解决 JavaScript 浮点数精度问题（如 0.1 + 0.2 = 0.3），结果最多显示 10 位有效数字
- **responsive-layout**: 响应式设计，移动端（≤480px）全屏布局，桌面端居中卡片式布局，按钮尺寸适配触摸和鼠标操作
- **error-handling**: 除以零时显示"错误"提示，无效表达式显示"格式错误"，不崩溃、不卡死

## Constraints

- 纯前端实现，无后端依赖
- 技术栈限定为 React + Tailwind CSS
- 表达式求值使用自实现解析器（递归下降或逆波兰），不使用 `eval()`
- 单次表达式最大长度 50 字符
- 运算结果超出 JavaScript `Number.MAX_SAFE_INTEGER` 时显示科学计数法
- 首屏加载时间 < 1 秒（无网络请求）

## Out of Scope

- 科学计算功能（sin、cos、log、幂运算等）
- 单位换算
- 程序员模式（进制转换）
- 图形绘制 / 函数可视化
- 多主题 / 暗黑模式切换
- 用户账户与云同步
- 括号与运算优先级（MVP 仅支持线性左到右计算）
