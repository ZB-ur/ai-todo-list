## Goal

构建一个基于 Web 的简洁计算器应用，支持基础四则运算，界面美观、交互流畅，使用 React + Tailwind CSS 实现。

## Features

- **basic-arithmetic**: 支持加（+）、减（−）、乘（×）、除（÷）四则运算，支持小数点输入，支持连续运算（如 3 + 5 × 2）
- **expression-display**: 顶部显示当前输入的完整表达式，底部显示实时计算结果或当前输入数字
- **clear-and-backspace**: 提供 AC（全部清除）按钮重置计算器状态，提供退格（⌫）按钮删除最后一位输入
- **keyboard-support**: 支持物理键盘输入数字和运算符（0-9、+、-、*、/、Enter、Backspace、Escape）
- **error-handling**: 除以零时显示错误提示（如"不能除以零"），溢出或非法输入时显示友好错误信息，错误状态下任意输入可恢复正常状态
- **responsive-layout**: 响应式布局，在桌面和移动端均可正常使用，按钮大小适合触屏点击（最小 48×48px 触摸目标）

## Constraints

- 使用 React + Tailwind CSS 实现，与项目现有技术栈保持一致
- 纯前端实现，不依赖后端服务或第三方 API
- 禁止使用 `eval()` 或 `Function()` 解析表达式，必须实现安全的数学表达式解析器
- 表达式解析器需正确处理运算符优先级（乘除优先于加减）
- 数字精度：使用 JavaScript 浮点数原生精度，对常见浮点误差（如 0.1 + 0.2）做显示层修正（最多保留 10 位有效数字）

## Out of Scope

- 科学计算功能（三角函数、对数、指数等）
- 计算历史记录
- 括号支持
- 单位换算
- 主题切换 / 暗黑模式
- 数据持久化或云同步
- 多语言国际化
