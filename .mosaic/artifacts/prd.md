## Goal

提供一个轻量、即开即用的 Web 计算器，支持基础四则运算，零学习成本，响应迅速。

## Features

- **F1 — 四则运算**：支持加、减、乘、除四种基本运算，包含小数点输入
- **F2 — 表达式显示**：屏幕上方显示完整输入表达式（如 `3 + 5 × 2`），下方显示当前输入或计算结果
- **F3 — 按钮面板**：标准计算器布局，包含数字键（0-9）、运算符键（+ − × ÷）、等号键、清除键（C/AC）、退格键、正负切换键（±）、百分号键（%）
- **F4 — 键盘输入**：支持物理键盘数字及运算符输入，Enter 等于计算，Backspace 退格，Escape 清除
- **F5 — 计算历史**：右侧或下方显示最近 20 条计算记录（表达式 = 结果），点击历史记录可将结果填入当前输入，存储于 localStorage
- **F6 — 按键反馈**：按钮点击/按下时有明确的视觉状态变化（hover、active），确保操作反馈即时可感知
- **F7 — 响应式布局**：桌面端居中卡片式布局，移动端全屏适配，按钮尺寸适合触控操作

## Constraints

- 纯前端实现，不依赖后端 API，所有计算逻辑在浏览器端完成
- 技术栈限定 React + Tailwind CSS
- 不使用 `eval()` 执行用户输入，必须实现安全的表达式解析
- 运算精度：使用合理策略处理浮点精度问题（如 `0.1 + 0.2` 应显示 `0.3`）
- 除以零时显示错误提示，不崩溃
- 浏览器兼容：支持最新版 Chrome、Firefox、Safari、Edge

## Out of Scope

- 科学计算功能（三角函数、对数、幂运算等）
- 程序员模式（进制转换、位运算）
- 图形绘制 / 函数绘图
- 单位换算、汇率转换
- 用户账号体系 / 云端同步
- 主题切换 / 深色模式
- 多语言国际化
