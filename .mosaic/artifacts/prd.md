## Goal
构建一个支持四则运算的 Web 计算器，提供清晰的按钮布局、键盘输入支持和优雅的错误处理，可在桌面和移动端流畅使用。

## Features
- **basic-arithmetic**: 支持加（+）、减（−）、乘（×）、除（÷）四则运算，用户可通过连续输入数字和运算符构建表达式并求值
- **display**: 顶部显示区域，实时显示当前输入的表达式和计算结果；支持显示上一步运算表达式
- **button-pad**: 标准计算器按钮布局，包含数字键（0-9）、运算符键（+ − × ÷）、小数点（.）、等号（=）、清除（C）、退格（⌫）键；按钮尺寸适配触控操作（最小 48×48px）
- **keyboard-input**: 支持物理键盘输入：数字键、运算符键（+、-、*、/）、Enter 键求值、Escape 键清除、Backspace 键退格
- **error-handling**: 优雅处理除零错误（显示"错误"提示）、连续运算符输入（自动替换最后一个运算符）、浮点精度问题（结果保留合理小数位数，消除 0.1+0.2≠0.3 类问题）、数值溢出提示
- **responsive-layout**: 响应式设计，桌面端居中展示合理宽度，移动端全宽布局，按钮自适应缩放，确保触控友好

## Constraints
- 纯前端实现，无需后端服务
- 技术栈：React + Tailwind CSS
- 浮点运算精度需特殊处理（避免 JavaScript 原生浮点误差），可使用 `toFixed` 或乘法缩放策略
- 表达式求值禁止使用 `eval()` 或 `Function()` 构造器，须使用安全的解析逻辑（如手写递归下降解析器或状态机）
- 最小可触控目标 48×48px（遵循 WCAG 触控目标指南）
- 支持主流浏览器（Chrome、Safari、Firefox、Edge 最新两个版本）

## Out of Scope
- 科学计算功能（三角函数、对数、幂运算等）
- 计算历史记录功能（MVP 后增强）
- 单位换算
- 多行/笔记式计算
- 图形/函数绘图
- 主题切换 / 暗黑模式
- 本地存储 / 数据持久化
- 国际化 / 多语言支持
