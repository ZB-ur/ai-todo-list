## Goal

提供一个轻量、快速、易用的 Web 端简单计算器，支持基础四则运算，具备良好的交互体验。

## Features

- **basic-arithmetic**: 支持加（+）、减（-）、乘（×）、除（÷）四则运算，支持连续运算（如 `3 + 5 × 2`），遵循标准运算优先级
- **decimal-support**: 支持小数点输入与小数运算，防止重复小数点输入
- **clear-and-backspace**: 提供"AC"全部清除按钮和"⌫"退格按钮，AC 重置为初始状态，退格删除最后一位输入
- **display**: 单行显示屏，实时显示当前输入的表达式和计算结果；数字超长时自动缩小字号或截断显示
- **keyboard-support**: 支持物理键盘输入：数字键（0-9）、运算符（+、-、*、/）、Enter 计算、Escape 清除、Backspace 退格
- **button-layout**: 标准 4×5 网格布局（0-9 数字、4 个运算符、小数点、等号、AC、退格），按钮有点击反馈（视觉高亮）
- **error-handling**: 除以零时显示"Error"提示，而非崩溃或显示 Infinity；点击 AC 可从错误状态恢复

## Constraints

- 纯前端实现，使用 React + Tailwind CSS，无后端依赖
- 表达式求值必须自行实现（禁止使用 `eval()`），防止代码注入
- 支持主流浏览器（Chrome、Firefox、Safari、Edge 最新两个版本）
- 移动端响应式适配，按钮最小点击区域 44×44px
- 首屏加载时间 < 1 秒（静态资源）

## Out of Scope

- 科学计算（三角函数、对数、幂运算等）
- 计算历史记录
- 主题切换 / 深色模式
- 括号与复杂表达式嵌套
- 单位换算、汇率转换
- 本地存储 / 数据持久化
- 多语言 / 国际化
